Detailed Description

This node is responsible for the ball detection, so implement all CV algorithm for recognize the ball, follow it and reache it.

In fact the node publish to 2 topics:

  • image_raw/compressed (send compressed image)
  • /ball_status And subscribe to:
  • image_raw/compressed
  • head_status (for controlling the head movement)
